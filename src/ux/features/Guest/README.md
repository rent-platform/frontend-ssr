# Guest Experience (Гостевой режим)

## Описание
Гостевой режим предоставляет неавторизованным пользователям ограниченный доступ к платформе с призывом к регистрации.

## Эндпоинт
`/dev-ui/guest`

## Возможности для гостей

### ✅ Доступно:
- Просмотр первых 12 объявлений из каталога
- Фильтрация по категориям
- Просмотр превью карточек товаров

### ❌ Ограничено:
- Детальный просмотр объявлений (показывается модальное окно с призывом к авторизации)
- Связь с владельцами
- Добавление в избранное
- Создание объявлений

## Особенности реализации

### Компоненты
- `GuestExperience.tsx` - основной компонент гостевого режима
- `GuestExperience.module.scss` - стили с градиентным фоном

### UX паттерны
1. **Sticky header** с кнопками "Войти" и "Регистрация"
2. **Hero секция** с описанием возможностей платформы
3. **Ограниченный каталог** - показывается только 12 товаров
4. **Overlay на карточках** - при наведении показывается замок и призыв войти
5. **Модальное окно** - при клике на карточку предлагается авторизоваться
6. **CTA блок** - внизу страницы призыв к регистрации

### Дизайн
- Градиентный фон (purple-blue)
- Glassmorphism эффекты
- Современные анимации через Framer Motion
- Полностью адаптивный дизайн

## Интеграция

### Импорт
```tsx
import { GuestExperience } from '@/ux/features';
```

### Использование в роутинге
```tsx
// src/app/dev-ui/guest/page.tsx
export default function GuestPage() {
  return <GuestExperience />;
}
```

## Переиспользуемые компоненты
- `CatalogCard` - из основного каталога
- `CategoryRail` - фильтр по категориям
- `CatalogSkeletonCard` - skeleton loader
